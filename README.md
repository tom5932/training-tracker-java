# Training Tracker Java Application

A console-based Java application for creating and managing workout routines and exercises.

## Features
- Menu-driven console interface
- Create and select training routines
- Add and view exercises
- Record workout history and completion status
- Uses object-oriented programming and ArrayLists

## Status
Work in progress. Core routine, exercise, and workout record flows are currently implemented.

## Technologies
- Java
- Git
- VS Code
